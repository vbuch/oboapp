# Agent Context

Canonical source for all AI agent primitives — instructions, prompts, agents, and skills.

[APM](https://microsoft.github.io/apm/) deploys these to `.github/` (Copilot) and `.claude/` (Claude Code) via `pnpm apm:install`.

For the full setup guide, see [docs/setup/apm-agent-resources.md](../docs/setup/apm-agent-resources.md).
